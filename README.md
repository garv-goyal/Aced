## App: ACED 
AI voice-based mock interview web app

**Features:**
- 3-step Guided Stepper UI (Role → Job Description → Resume)
- AI generates 5 targeted interview questions (GPT-4)
- Voice interview: browser SpeechSynthesis TTS + Web Speech API STT
- Visual scorecard: Clarity, STAR method, Sentiment scores

**Design:**
- Blue/white color scheme with subtle CSS grid line background
- BrainCircuit logo icon (lucide-react)
- Slide-in history drawer
- Animated step transitions
